DEFINITIONS

Phrase Phrase is defined loosely here as a phrase in it's regular sense (a group of words that makes sense), an expression, an idiom, or even a complete sentence.
Keywords Must be a part of the phrase & should be an important word or a group of words. You will search phrases by keywords. Keywords are also used to build test questions
Labels Using labels to organize your phrases into categories. You can search phrases by labels.
Phrase Notes Notes can be using context or description of the phrase.
Mastery
  • Learning: Indicates that you are still learning the phrase.
  • Mastered: Indicates that you have mastered the phrase.
Language Phrase language

MANAGE BACKUP

Phrase Builder supports two options of data backup/restore:

  • Android Backup Service
  • External Storage Backup
Android Backup Service VS External Storage Backup
Your backup data is on Google Server Your backup data is on local storage (SD card or Internal Storage)
Backup is requested automatically (weekly) You have to manually perform your backup
Backup delayed - You don't know when Android perform backup for your app data Instant backup - You can backup your app data anytime.
Android Backup Service
  • Please make sure Back up my data & Automatic restore are enabled on your phone (Settings > Backup & reset). A backup request does not result in an backup immediately performed. Instead, Android will wait until an appropriate time to perform the backup.
  • Phrase Builder will notify you whenever a backup is requested or a backup is performed. You will see a notification message in your device notification area.
  • If you NEVER see a backup performed notification message, either Android Backup Service is NOT available on your device or it DOESN'T work properly. In this case, please use External Storage Backup!
Backup day The day that Android will request a backup
  • Sunday
  • Monday
  • Tuesday
  • Wednesday
  • Thursday
  • Friday
  • Saturday
Backup time The time that Android will request a backup
Last backup timestamp Last backup timestamp if any
Warning!!! Please make sure that there is a cloud backup before resetting your device.
External Storage Backup
Backup location Always backup your app data to a fixed location: External Storage root directory / phrase_builder_data.bak
Last backup timestamp Last backup timestamp if any
Backup button Perform a backup. Please note that the backup will override any existing backup!
Restore button Perform a restore. Please note that the restore will override any existing app data!
Warning!!! Please make sure that you have a copy of phrase_builder_data.bak in your computer/usb, etc. before resetting your device.

USING TIPS

Search Phrases Keywords/Label Input
  • No accent marks required
  • No minus '-' required
  • Example 1: nha hang is identical to nhà hàng
  • Example 2: nha hang is identical to nha-hang
Input Labels Label Input
  • No accent marks required
  • No minus '-' required
  • Example 1: khach san is identical to khách sạn
  • Example 2: khach san is identical to khach-san
